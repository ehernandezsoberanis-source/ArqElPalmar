# Arq El Palmar

Sitio editorial premium construido con Next.js App Router, TypeScript, Tailwind CSS y Framer Motion.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Tema

- Tema claro fijo.

### Ajuste de color (tokens)

Todos los tokens principales viven en [app/globals.css](./app/globals.css):

- `--color-paper`, `--color-paper-alt`, `--color-ink`
- `--color-muted-1..4`
- `--color-border`, `--color-border-strong`
- `--color-card-tint`, `--color-map-shell`
- `--shadow-header`, `--shadow-card-*`, `--shadow-project-*`
- `--service-gradient`

Para ajustar el look:

1. Edita los tokens en `:root`.
